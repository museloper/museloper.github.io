휴고 블로그 시작하기

By museloper | July 28, 2021 | 3 minutes

본 게시물은 윈도우10 환경을 기준으로 작성 되었습니다.


1. 설치

1. 디렉토리 생성

  1. 메인 디렉토리를 생성합니다. C:\Hugo

  2. 서브 디렉토리를 생성합니다. C:\Hugo\bin, C:\Hugo\Sites


2. 배포파일 다운로드

패키지 매니저를 사용하실 줄 아시는 분이라면 Chocolatey 혹은 Scoop를 통해서 설치하는 것을 권장합니다.

패키지 매니저에 대해서 친숙하지 않으시다면 깃허브에 공식 배포된 휴고 파일을 다운 받으시면 됩니다!

👉 휴고 깃허브 릴리즈 👈

Sass/SCSS 사용을 원하신다면 hugo_extended 버전을 다운 받으셔야 합니다!

파일을 다운받은 후 압축을 해제하여 hugo.exe 파일을 미리 생성해둔 C:\Hugo\bin 디렉토리 하위로 이동시킵니다.


3. 환경변수 설정

  1. 내 PC 에서 마우스 우클릭 하여 속성 클릭


  1. 고급 시스템 설정 클릭


  1. 고급 탭에서 환경 변수 클릭


  1. 시스템 변수 카테고리에서 Path를 선택한 후 편집 버튼 클릭


  1. 새로운 라인에 C:\Hugo\bin 추가하고 확인


4. 버전 확인 

1
2

> hugo version
hugo v0.82.0-9D960784+extended windows/amd64 BuildDate=2021-03-21T17:28:04Z VendorInfo=gohugoio

3. 테마 추가

👉 https://themes.gohugo.io/ 👈 에서 다양한 테마를 찾을 수 있습니다.

원하는 테마를 찾았다면 생성한 사이트에 테마를 추가해야 합니다.

1
2
3
4
5
6
7

> cd sample
rem 생성한 사이트 디렉토리로 이동합니다.

> git init
> git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
rem 각 테마는 원작자의 깃헙 저장소에 공개되어 있습니다. 
rem 깃헙 저장소에서 테마 디렉토리로 서브모듈을 추가합니다.


4. 테마 연결

생성된 사이트가 어떤 테마를 사용할 것인지 명시해줘야 합니다.

아래와 같이 명령어로 테마를 연결합니다.

1
2

> echo theme = "ananke" >> config.toml
rem config.toml 파일에 theme = "ananke" 을 추가하는 명령어

혹은 config.toml 파일을 직접 열어 theme = "ananke"를 추가하셔도 됩니다.

서버가 켜지면 http://localhost:1313/ 주소를 통해 빌드된 사이트에 접근할 수 있습니다!

하지만 아직 아무런 컨텐츠가 없기 때문에 사이트에 접근해도 휑하게 느껴지네요 ^^;; 휑한 사이트를 채워줄 컨텐츠를 생성해 봅시다.

6. 컨텐츠 생성

아래와 같이 hugo new ***.md 명령어를 통해 컨텐츠를 생성할 수 있습니다.

1
2

> hugo new posts/sample.md
c:\Hugo\Sites\sample\content\posts\sample.md created

생성된 컨텐츠의 경로는 content 디렉토리의 하위로 들어가게 됩니다.

휴고 블로그의 컨텐츠들은 .md 파일 즉 마크다운 문법으로 작성되는 파일 입니다. 마크다운 문법이 생소하시다면 마크다운 문법 포스팅을 참조하세요 :)


7. 테마 커스터마이징

config.toml 파일을 열고 내용을 수정합니다.

참고로 테마마다 .toml 파일을 사용할 수도 있고 .yaml 파일을 사용할 수도 있습니다. 저마다의 문법이 조금씩 다르니 유의하셔야 합니다.

1
2
3
4

baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

themes/ananke/exampleSite 디렉토리를 참조하세요!!
